# Multilingual-Bot-Assistant
**Multilingual AI Chatbot System**  Developed a multilingual AI-powered chatbot with semantic search capabilities using Sentence Transformers. The system supports multiple languages, provides intelligent FAQ responses, suggestion prompts, and a modern user-friendly interface built with Flask and Python.

🤖 Multilingual AI FAQ Chatbot

An AI-powered multilingual chatbot designed to provide instant answers to frequently asked questions in multiple languages. The system uses Natural Language Processing (NLP) and semantic similarity matching to understand user queries and deliver accurate responses through an intuitive web interface.

🌟 Features

🌐 Multilingual Support
• Supports English, Hindi, Marathi, Gujarati, Bengali, and Hinglish
• Language-specific suggestions and responses
• User-friendly language switching

🧠 AI-Powered Question Answering
• Semantic search using Sentence Transformers
• Intelligent FAQ matching based on query meaning
• Handles variations of the same question effectively

💬 Interactive Chat Interface
• Modern responsive chatbot UI
• Real-time messaging experience
• Typing indicators and quick suggestion buttons
• Mobile-friendly design

📚 Educational Information System
• Academic FAQs (timetables, exams, placements)
• Admission and registration guidance
• Fee, scholarship, and attendance information
• Career, internship, and placement support

📞 Support & Assistance
• Fallback responses for unknown queries
• Direct WhatsApp and call support integration
• Easy access to management assistance

🛠️ Technologies Used
• Python

• Sentence Transformers
• PyTorch
• NLP (Natural Language Processing)
• HTML5

• JavaScript

🎯 Key Highlights
